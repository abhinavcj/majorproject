HOW TO RUN: SignBridge Interactive Platform
(Sign Language Detection, Speech Assist & Text-to-ASL)

--- STEP 1: Start the Backend Server ---
The backend powers the machine learning sign-detection and the text-to-ASL video routing.

1. Open your terminal.
2. Navigate to the project folder:
   cd /Users/abhinav/Downloads/Sign-Language-To-Text-and-Speech-Conversion
3. Activate the virtual environment:
   source venv/bin/activate
4. Run the FastAPI server:
   uvicorn backend_api:app --host 0.0.0.0 --port 8000

Wait until you see "Model Loaded Successfully!" before proceeding.

--- STEP 2: Start the Frontend UI ---
1. Open a *new, separate* terminal window.
2. Navigate to the project folder:
   cd /Users/abhinav/Downloads/Sign-Language-To-Text-and-Speech-Conversion
3. Start a local static file server:
   python3 -m http.server 5500

--- STEP 3: Access the Application ---
Open your web browser and navigate to the features:
▶ Main App (Sign to Text Translation): http://localhost:5500/index.html
▶ Speech Assist (Voice to Deaf Communicator): http://localhost:5500/speech.html
▶ Text to Sign Language (Text to ASL Video Generation): http://localhost:5500/text_to_sign.html

(You can seamlessly navigate between these using the top navigation bar!)

--- TROUBLESHOOTING: Ports Already in Use ---
If you get an error saying the address is already in use, you can kill the background processes with these commands:

Kill Backend (Port 8000):
kill $(lsof -ti :8000)

Kill Frontend (Port 5500):
kill $(lsof -ti :5500)

After killing them, you can restart both servers fresh.
